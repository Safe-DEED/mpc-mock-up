This platform uses a trusted third party (TTP) for secure data contribution. Your input will be used to compute operational performance metrics by comparing your data with data from other distribution centers also making use of this platform.


Please be ensured that your input is kept confidential. The TTP is a trusted entity which provides secure processing of your data. Only the output of the computation is disclosed to you and other participants. The computation functions used for the analytics can be viewed here.

The complete data flow is shown in the illustration above. The steps performed here are uploading your input data, viewing your input data, and submitting your input data.

A template is available to make inputting your data more easy. The template can be downloaded by clicking on Download Template button below. The purpose of this template is to allow you to gather the input of the data offline and to allow you to keep your input data available on your local machine for later.
