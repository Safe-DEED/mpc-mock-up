DESCRIPTION

Welcome! This demonstration tool is built to guide you though the technological and business aspects of multi party computation (abbreviated herein as MPC). The tool is comprised of three modules. In the first module we will describe data sharing and the context in which MPC is discussed which is supply chain. In the second module we will describe the basics of MPC. After this module you should have a basic understanding of the MPC building blocks. In the third module we describe a specific use-case. The aim of this module is to show how MPC can be used in practice. A mock-up (not a real working application) is used to demonstrate the MPC application. It provides you with the look and feel of an MPC application. Please start by clicking on module 1 below.

Note: if your browser has default settings (javascript enabled), then you will be guided automatically through these modules.
